# Ladder of Inference Flow

Argyris ladder: separate observable data from interpretations, assumptions, and conclusions.

## When to use

- Heated disagreement ("they're wrong" vs "they're evil")
- Post-incident blame
- When conclusions feel obvious but unstated steps are hidden

## The ladder (bottom → top)

1. **Observable data** — what was actually seen/heard (no interpretation)
2. **Selected data** — what did we pay attention to? What did we ignore?
3. **Meanings** — what labels did we apply?
4. **Assumptions** — what did we take for granted?
5. **Conclusions** — what did we decide?
6. **Beliefs** — how does this reinforce wider beliefs?
7. **Actions** — what will we do as a result?

## Steps

1. **State the conclusion** — what action or belief is in dispute?
2. **Walk down the ladder** — document each rung explicitly
3. **Find leaps** — where did we jump without evidence?
4. **Replay** — rebuild from observable data only; does the same conclusion follow?
5. **Alternative ladders** — how would a reasonable person climb differently?
