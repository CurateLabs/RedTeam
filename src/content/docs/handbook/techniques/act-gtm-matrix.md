## Summary

Table matching tools to ACT, GTM, or both — pick and sequence by situation.

## When to use

Planning which tools to run and in what order.

## Method

1. Classify decision risk and time.
2. Mark ACT vs GTM gap.
3. Select tools from matrix.
4. Sequence diverge before converge.

## RedTeam notes

Category: **facilitation**. RedTeam may extend or alias this technique.
