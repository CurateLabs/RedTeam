## Summary

Explore consequences when a key variable changes.

## When to use

Sensitivity to one driver (price, regulation, competitor action).

## Method

Name variable → set alternative values → trace implications for plan.

## RedTeam notes

Category: **divergent**. RedTeam may extend or alias this technique.
