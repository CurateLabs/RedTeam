## Summary

Sequential "what if" chain along a narrative thread.

## When to use

Exploring cascading consequences of a decision.

## Method

State event → what if? → next pearl → continue chain → identify breakpoints.

Category: **divergent**. RedTeam may extend or alias this technique.
