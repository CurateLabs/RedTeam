**Command:** `/redteam assumptions`

## Summary

List what must be true for the plan to work; flag critical unvalidated beliefs.

## When to use

30–60 min before resource commitment.

## Method

Extract ≥10 assumptions → classify → mark validated/unvalidated → test critical ones.

## RedTeam notes

Category: **analytical**. RedTeam may extend or alias this technique.
