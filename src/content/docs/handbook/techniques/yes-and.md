## Summary

Build on prior ideas without blocking — improv rule for ideation phases.

## When to use

Divergent phase only; team habitually says "yes, but."

## Method

Each response must start by extending prior idea before adding new angle.

## RedTeam notes

Category: **divergent**. RedTeam may extend or alias this technique.
