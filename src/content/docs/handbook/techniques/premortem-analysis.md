## Summary

Assume failure; work backward to causes and preventions.

## When to use

30–60 min; optimism bias; pre-launch.

## Method

Future failure scenario → brainstorm causes → rank → prevent/detect/respond for top risks.

## RedTeam notes

Category: **adversarial**. RedTeam may extend or alias this technique.
