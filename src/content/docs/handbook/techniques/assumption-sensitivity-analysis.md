## Summary

Rank assumptions by what happens if each is wrong (collapse vs. absorb).

## When to use

Prioritizing which assumptions to test first.

## Method

Rate sensitivity (collapse/degrade/absorb) × confidence → focus on collapse + low confidence.

## RedTeam notes

Category: **analytical**. RedTeam may extend or alias this technique.
