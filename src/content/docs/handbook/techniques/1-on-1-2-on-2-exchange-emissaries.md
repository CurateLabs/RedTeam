## Summary

Subgroup dialog with emissaries cross-pollinating between groups.

## When to use

45–90 min; need more divergence than 1-2-4; breaking silos.

## Method

Individual write → 1:1 → 2:2 → emissary to other group → return with feedback.

## RedTeam notes

Category: **facilitation**. RedTeam may extend or alias this technique.
