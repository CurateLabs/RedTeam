# Argument Deconstruction

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam challenge`

## Summary

Separate premises, inference, and conclusion; test each link.

## When to use

Persuasive narrative masking weak logic.

## Method

Diagram: premise → therefore → conclusion. Test each arrow independently.

## RedTeam notes

Category: **analytical**. RedTeam may extend or alias this technique.
