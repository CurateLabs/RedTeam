# TRIZ

**Chapter:** [Chapter 8 — Tools, Techniques & Practices](../chapters/08-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam ideate`

## Summary

Systematic inventive problem-solving from contradictions (Altshuller).

## When to use

45–90 min; engineering trade-offs; contradictory requirements.

## Method

State contradiction → TRIZ principles → generate inventive solutions.

## RedTeam notes

Category: **cultural**. RedTeam may extend or alias this technique.
