# Polarity Mapping

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** Extension
**Command:** `/redteam frame`

## Summary

Manage ongoing tensions (poles) instead of false either/or solutions.

## When to use

Persistent speed vs quality style debates.

## Method

Name poles → map upsides of each → plan to hold both.

## RedTeam modifications

Category: **extension**. RedTeam may extend or alias this technique.
