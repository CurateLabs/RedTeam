# 5 Whys

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam 5-whys`

## Summary

Iterative why-questions to reach root causes and hidden assumptions.

## When to use

15–30 min; incident; recurring symptom fixes.

## Method

State problem → why? ×5 → validate root → distinguish symptom vs cause.

## RedTeam modifications

Category: **analytical**. RedTeam may extend or alias this technique.
