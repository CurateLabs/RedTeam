# Alternative Futures Analysis

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam futures`

## Summary

Multiple plausible futures against which to stress-test the plan.

## When to use

60–120 min; strategic planning; high uncertainty.

## Method

Pick 2 uncertainties → 4 scenarios → test plan in each → find robust moves and signposts.

## RedTeam notes

Category: **strategic**. RedTeam may extend or alias this technique.
