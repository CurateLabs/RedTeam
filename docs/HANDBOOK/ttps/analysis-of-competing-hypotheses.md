# Analysis of Competing Hypotheses (ACH)

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam ach`

## Summary

Matrix scoring evidence against mutually exclusive hypotheses.

## When to use

60–180 min; multiple explanations; confirmation bias risk.

## Method

List hypotheses → list evidence → mark C/I/N per cell → fewest inconsistencies wins.

## RedTeam modifications

Category: **analytical**. RedTeam may extend or alias this technique.
