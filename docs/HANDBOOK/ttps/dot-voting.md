# Dot Voting

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** GTM
**Command:** `/redteam converge`

## Summary

Allocate limited votes across options for quick prioritization.

## When to use

10 min; too many options; need democratic convergence.

## Method

Each person gets N dots; place on options; count; discuss ties and surprises.

## RedTeam notes

Category: **facilitation**. RedTeam may extend or alias this technique.
