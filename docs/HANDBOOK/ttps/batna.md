# BATNA

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam converge`

## Summary

Best Alternative to a Negotiated Agreement — know your walk-away before negotiating.

## When to use

30–45 min; deal approval; vendor selection.

## Method

List alternatives if no deal → improve best → compare to offer → never accept worse than BATNA.

## RedTeam modifications

Category: **adversarial**. RedTeam may extend or alias this technique.
