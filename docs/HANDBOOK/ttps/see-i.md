# SEE-I

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam challenge`

## Summary

State, Elaborate, Exemplify, Illustrate — clarify before debate.

## When to use

10–15 min; people argue past each other on undefined terms.

## Method

State term → elaborate → give example → illustrate (analogy/diagram).

## RedTeam modifications

Category: **analytical**. RedTeam may extend or alias this technique.
