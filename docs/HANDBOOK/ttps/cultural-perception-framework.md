# Cultural Perception Framework

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam culture`

## Summary

How different cultural actors perceive the same events.

## When to use

Multi-stakeholder conflict; incompatible narratives.

## Method

Per actor: perception, values, fears, goals → compare maps.

## RedTeam modifications

Category: **cultural**. RedTeam may extend or alias this technique.
