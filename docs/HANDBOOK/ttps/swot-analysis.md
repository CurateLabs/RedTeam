# SWOT Analysis

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam converge`

## Summary

Strengths, Weaknesses, Opportunities, Threats for an option or entity.

## When to use

30–60 min; option comparison; strategic snapshot.

## Method

Fill four quadrants → cross-link (SO/WO/ST/WT strategies) → prioritize.

## RedTeam modifications

Category: **strategic**. RedTeam may extend or alias this technique.
