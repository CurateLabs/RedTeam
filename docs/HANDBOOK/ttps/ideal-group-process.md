# Ideal Group Process

**Chapter:** [Chapter 7 — Tools, Techniques & Practices](../chapters/07-tools-techniques-practices.md)
**ACT/GTM:** Both
**Command:** `/redteam groupthink`

## Summary

Diverge → analyze → debate → converge loop with rotating ACT/GTM tools.

## When to use

60–180 min group sessions on complex decisions.

## Method

Cycle through divergent tools, ACT analysis, structured debate, convergent tools until time ends.

## RedTeam modifications

Category: **facilitation**. RedTeam may extend or alias this technique.
