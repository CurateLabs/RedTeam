# Analogy Suitability Analysis

**Chapter:** [Chapter 8 — Tools, Techniques & Practices](../chapters/08-tools-techniques-practices.md)
**ACT/GTM:** ACT
**Command:** `/redteam challenge`

## Summary

Test whether the analogy in use actually fits.

## When to use

Plan justified by "it's like Uber for X."

## Method

Map source and target domains; list matches and mismatches; decide if analogy helps or misleads.

## RedTeam notes

Category: **analytical**. RedTeam may extend or alias this technique.
