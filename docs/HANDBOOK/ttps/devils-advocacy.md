# Devil's Advocacy

**Chapter:** [Chapter 8 — Tools, Techniques & Practices](../chapters/08-tools-techniques-practices.md)
**ACT/GTM:** Both
**Command:** `/redteam devils-advocate`

## Summary

Structured strongest-case against the prevailing view.

## When to use

20–45 min; consensus too fast; launch approval.

## Method

Steel-man prevailing view → structured case against → assess fatal vs addressable objections.

## RedTeam notes

Category: **adversarial**. RedTeam may extend or alias this technique.
