# Inversion

**Chapter:** [Chapter 8 — RedTeam Extensions](../chapters/08-redteam-extensions.md)

**Command:** `/redteam invert`

## Summary

Munger-style: what would guarantee failure? Flip into avoidance rules.

## When to use

Strategy, hiring, process design; fast adversarial pass.

## Method

List failure guarantors → invert to constraints → audit current plan.
