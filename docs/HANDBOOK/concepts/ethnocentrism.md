# Ethnocentrism

**Chapter:** [Chapter 3 — Cultural Empathy](../chapters/03-cultural-empathy.md)

## Summary

Judging other cultures by the standards of one's own — the primary bias cultural empathy counteracts.

## When to use

When analysis assumes "normal" is universal; when stakeholders are labeled irrational.

## Method

Name the home-frame explicitly. Ask what behavior looks like from inside the other frame.
