# Kluckhohn's Six Dimensions

**Chapter:** [Chapter 3 — Cultural Empathy](../chapters/03-cultural-empathy.md)

## Summary

Age-old cultural variation axes for comparing cultural defaults (human nature, activity, time, relational, etc.).

## When to use

Comparing team or stakeholder cultural defaults without reducing to nationality alone.

## Method

Score or discuss each dimension for each actor; look for mismatches driving conflict.
