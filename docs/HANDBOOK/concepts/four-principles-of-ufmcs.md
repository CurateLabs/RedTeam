# Four Principles of UFMCS

**Chapter:** [Chapter 1 — Foundational Concepts](../chapters/01-foundational-concepts.md)

## Summary

The pillars every Red Team practice rests on: ACT, Cultural Empathy, GTM, and Self-Awareness.

## When to use

When designing any red team session, command flow, or facilitator checklist.

## Method

1. **ACT** — make reasoning explicit.
2. **Cultural Empathy** — include missing worldviews.
3. **GTM** — protect dissent and alternatives.
4. **Self-Awareness** — know your own biases first.
