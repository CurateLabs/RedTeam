# Four Core Principles

**Chapter:** [Chapter 1 — Foundational Concepts](../chapters/01-foundational-concepts.md)

## Summary

ACT, cultural empathy, GTM, and self-awareness — the organizing frame for all commands.

## When to use

Designing any red team session or command flow.

## Method

1. **ACT** (Applied Critical Thinking) — explicit reasoning.
2. **Cultural empathy** — understand missing worldviews and communication frames.
3. **GTM** (Groupthink Mitigation) — safe dissent.
4. **Self-awareness** — know your biases first.

## RedTeam notes

Civilian adaptation of source-handbook techniques. We update and extend over time — not verbatim doctrine.
