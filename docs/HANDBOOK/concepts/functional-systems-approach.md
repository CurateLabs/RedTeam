# Functional Systems Approach

**Chapter:** [Chapter 3 — Cultural Empathy](../chapters/03-cultural-empathy.md)

## Summary

Analyze how cultural subsystems interact functionally rather than as isolated lists.

## When to use

When single-factor explanations fail in complex social systems.

## Method

Model feedback between political, economic, social, and information subsystems.
