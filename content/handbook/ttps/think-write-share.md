# Think-Write-Share

**Chapter:** [Chapter 8 — Tools, Techniques & Practices](../chapters/08-tools-techniques-practices.md)
**ACT/GTM:** GTM
**Command:** `/redteam groupthink`

## Summary

Silent individual write before any discussion — defeats anchoring.

## When to use

10–20 min; first speaker bias; hierarchical rooms.

## Method

Prompt → silent write → pair share → optional full group.

## RedTeam notes

Category: **facilitation**. RedTeam may extend or alias this technique.
