# Onion Model

**Chapter:** [Chapter 3 — Cultural Empathy](../chapters/03-cultural-empathy.md)
**ACT/GTM:** ACT
**Command:** `/redteam frame`

## Summary

Culture manifests in layers: artifacts → espoused values → underlying assumptions.

## When to use

Understanding why stated values don't match behavior; org culture diagnosis.

## Method

Peel from visible artifacts inward. Ask what assumptions would make observed behavior rational.
