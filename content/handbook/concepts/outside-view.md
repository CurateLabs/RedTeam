**Chapter:** [Chapter 8 — Tools, Techniques & Practices](../chapters/08-tools-techniques-practices.md)

## Summary

Base-rate thinking: what happened in the reference class of similar past attempts?

## When to use

Forecasts, timelines, budgets, "this time is different" claims.

## Method

Name reference class → base rates → justified adjustments → compare to inside view.
