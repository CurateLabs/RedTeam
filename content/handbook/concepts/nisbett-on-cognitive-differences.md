# Nisbett on Cognitive Differences

**Chapter:** [Chapter 3 — Cultural Empathy](../chapters/03-cultural-empathy.md)

## Summary

How cognitive styles differ in attention, causality, and holism — affects planning and negotiation assumptions.

## When to use

Cross-border teams, analytic vs. holistic reasoning clashes.

## Method

Ask whether disagreement is data vs. framing. Test if parties seek root cause vs. situational balance.
