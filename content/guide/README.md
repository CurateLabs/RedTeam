## What RedTeam is

**RedTeam** is the skill name. It implements ***The Red Team Handbook*, Version 10** — a civilian successor to the defunct UFMCS v9.0 work. The handbook is [published separately](../handbook/) as reference content, not as developer documentation.

## Not penetration testing

If you need cybersecurity red teaming, this is the wrong tool. RedTeam is for **decisions, plans, strategies, policies, and arguments** — applied critical thinking before commitment.

## Need the theory?

Browse the [Handbook](../handbook/) for chapters on cultural empathy, groupthink mitigation, ACT/GTM techniques, and AI-assisted red teaming.
